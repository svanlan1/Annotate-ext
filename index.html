What is Marker?

Marker is a Google Chrome extension developed to help expedite making annotations on designs for web accessibility.

How to use Marker

Click on the 'Marker' icon in the upper right hand corner of Chrome

The 'Marker Panel' will appear on the left side of your screen by default. However, the panel is moveable and the position of the panel is stored. That means that wherever you drop then panel is where it will continue to appear each time you start Marker, until you move it again.

Each icon provides you with a different option.

  • - Draw a box on the screen
  • - Place a marker on the screen
  • - Removes all markers and boxes on the screen and resets the counter to zero
  • - Appends your recommendations to the bottom of the page

You can change the size and color of the box from the Marker Panel on any web page.

Changing the size and color of the box from the Marker Panel only changes the settings for that individual session.
To change the default color and box width, change the settings on the 'Options' page.

You can also change the 'Marker' or 'Pin' color.

Just like the box width and color, you can change the color on the 'Marker Panel' for that individual session, or change the default marker/pin color on the options page.

Why was Marker created?

After spending far too much time making annotations on designs and developed pages, writing the same text over and over and over and over again, I decided to create this extension to make this process easy, painless, consistent, and accurate.
Using Marker ensures that the latest guidelines are followed and allows the user to quickly create documents to hand off to developers.

How does Marker work?

With Marker, you are able to place pins on the page, highlight elements, and make annotations directly on the page.

Who should use Marker?

Marker can be used by anyone! Designers, developers, business analysts, or even less technically saavy users (e.g., I want someone to build me a website, but I want to make sure that it's accessible).

What types of pages does Marker work on?

Currently, Marker only works on HTML pages. You do not need to be online in order to use Marker.

What am I supposed to do with the annotations?

Once you've finished drawing your annotations on the page, press 'Save'. This saves your notes and draws them on the bottom of the 'body' element.
From here, for the foreseeable future, you can save the page by pressing 'Ctrl + S' to save the entire webpage. This saves all of the notes and annotations in HTML format that you can then send along.

What Accessibility Guidelines does Marker conform to?

Marker uses WCAG 2.0 Level AA recommendations. This is the most common set of guidelines followed by Web accessibility professionals.
